# POSIUFO Website Redesign - Make it ALIVE!

## Design Guidelines

### Design References
- **Streetwear/Creative brands**: Bold, playful, youthful energy
- **Style**: Modern, vibrant, animated, playful with dark accents

### Color Palette
- Primary: #000000 (Black - header/footer/buttons)
- Secondary: #1a1a2e (Dark navy)
- Accent 1: #e94560 (Hot pink/red - CTAs)
- Accent 2: #7c3aed (Purple - highlights)
- Accent 3: #f59e0b (Amber/Gold - stars/badges)
- Background: #faf5ff (Light lavender)
- Card BG: #ffffff (White)
- Text: #333333 (Dark gray)
- Muted: #9ca3af (Gray)

### Typography
- Headings: 'Poppins', sans-serif (bold, uppercase)
- Body: 'Inter', sans-serif (clean, readable)

### Key Component Styles
- Buttons: Black bg, white text, rounded-full, hover scale + shadow
- Cards: White bg, purple border on hover, rounded-xl, shadow-lg
- Sections: Full-width with generous padding
- Animations: Fade-in on scroll, hover scale, marquee text
